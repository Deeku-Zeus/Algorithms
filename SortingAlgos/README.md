# Sorting Algos
- Selection Sort (**O(n^2)**)
- Bubble Sort (**O(n^2)**)
- Insertion Sort (**O(n^2)**)
- Heap Sort (**O(n log(n))**)
- Quick Sort (**O(n^2)**)
- Merge Sort (**O(n log(n))**)
- Bucket Sort (**O(n^2)**)
- Radix Sort (**O(nk)**)
- Count Sort (**O(n+k)**)
- Shell Sort (**O(n^2)**)
- Tim Sort (**O(n log(n))**)
- Tree Sort (**O(n^2)**)
- Cube Sort (**O(n log(n))**)
